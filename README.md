# Golang-in-ACM

为方便算法竞赛选手在各大OJ刷题而开发的基础使用模板

将会优化 Go 的刷题体验，尽可能接近 C++ 甚至 Python

模板在 standard 文件夹下

# 使用说明

只需要将代码写入 Solve 函数即可

如果需要读入多组数据，请将 main 函数中的 T := 1 改为 T := 0

函数都有写注释，可以自行查看，目前有以下函数（部分实现了 64 位版本）：

- scan 读入（快速）
- print 输出（快速）
- println 换行输出（快速）
- ri 读入一个整数
- rs 读入一个字符串
- min 最小值
- max 最大值
- sum 求和
- reverse 翻转切片
